you can create gatsby app very easy
just you need npm, node and some internet for get dependencies

##first
check your node version greater than 11
```shell script
node --version
```
##second
install gatsby
```shell script
npm install gatsby
```
##third
create boilerplate and install dependencies
```shell script

```
